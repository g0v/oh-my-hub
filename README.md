Oh My Hub
=========

## Gulp

- pre-dev:
    - install: node
    - install: ruby 2.0.0 (use rubyuinstaller on windows, use rvm install 2.0.0 on - linux/mac)
    - install compass (gem install compass)
- development:
    - gulp watch
    - gulp open 
- deploy:
    - gulp deploy

## Hubs

- Projects Hub :: 專案列表
- Resource Hub :: 可自由利用的插圖，音樂等等
